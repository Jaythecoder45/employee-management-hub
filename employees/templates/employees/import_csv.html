Import Guidelines

  • The first row of the CSV file must be headers.
  • Expected headers: First Name, Last Name, Email, Phone, Job Title, Department, Salary, Hire Date, Status
  • Salary: Enter raw numbers (e.g. 75000 or 68500.50). Do not include currency symbols.
  • Hire Date: Must use the format YYYY-MM-DD (e.g. 2026-03-15).
  • Department: Must be one of: Engineering, Sales, HR, Marketing, Finance, Operations.
  • Status: Must be one of: Active, Inactive, On Leave.
